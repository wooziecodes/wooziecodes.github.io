# PDF Assets for OTR Copilot Project

This folder contains downloadable PDF documents for the website.

## Required File

**otr-capstone-report.pdf**
- Your complete INTS402 Capstone Project Final Report
- Ensure the PDF is the final, proofread version
- Recommended: Add bookmarks/outline for easier navigation
- File size: Aim for < 10MB (compress if needed)

## Adding the PDF

1. Save your final capstone report as `otr-capstone-report.pdf`
2. Place it in this directory: `/assets/pdf/`
3. The download link is already configured in:
   - `research.html` (line 145)
   - `otr-copilot.html` (in the links section)

## PDF Optimization

If your PDF is large (> 10MB), consider:

**Mac Preview:**
1. Open PDF in Preview
2. File > Export
3. Select "Quartz Filter: Reduce File Size"

**Adobe Acrobat:**
1. File > Save As Other > Reduced Size PDF

**Online Tool:**
- https://www.ilovepdf.com/compress_pdf

## Verify After Upload

Test the download link works:
1. Navigate to your website
2. Click "Download Full Report (PDF)"
3. Verify it downloads correctly
4. Check PDF opens properly
